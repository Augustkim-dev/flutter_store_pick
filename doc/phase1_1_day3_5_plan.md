## 🗓 Day 4: 메인/상세 화면 개선 및 표시

### 목표

### 오전 작업 (4시간)

#### 1. ShopCard 위젯 개선 (2시간)
```dart
// shop_card.dart 리팩토링
- 상점 유형 뱃지 추가
- 영업 상태 실시간 표시 (영업중/휴무/점심시간)
- 주력 브랜드 로고 표시
- 평점 및 리뷰 수 강조
- 당일배송/픽업 가능 아이콘
```

#### 2. 홈 화면 필터링 강화 (2시간)
```dart
// home_screen.dart 개선
- 상점 유형별 필터 (오프라인/온라인/복합)
- 브랜드별 필터
- 카테고리별 필터
- 편의시설 필터 (주차, 시착, 휠체어 등)
```

### 오후 작업 (4시간)

#### 1. ShopDetailScreen 대폭 개선 (3시간)
```dart

- 정보 탭 구조 추가:
  * 기본정보 탭
  * 영업정보 탭
  * 브랜드/카테고리 탭
  * 리뷰 탭
  * 공지사항 탭
- 상점 유형별 조건부 정보 표시
- 실시간 영업 상태 표시
```

#### 2. 새로운 위젯 생성 (1시간)
