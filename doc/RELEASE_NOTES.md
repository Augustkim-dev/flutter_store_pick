# Release Notes - 발레플러스 (Ballet+)

## Version 1.1.0 - 2025년 1월 20일

### 🎆 상점관리자 기능 대폭 강화!

발레플러스를 이용해 주시는 상점 사장님들을 위한 대규모 업데이트가 진행되었습니다.
이제 더욱 풍부하고 상세한 상점 정보를 관리하고 고객님께 제공할 수 있습니다.

---

### 🆕 주요 신규 기능

#### 🏢 상점 정보 관리 전면 개편

##### 탭 기반 편집 화면
상점 정보를 더욱 체계적으로 관리할 수 있도록 탭 구조로 전면 개편했습니다.

- **기본 정보 탭**: 상점명, 설명, 연락처, 사업자 번호
- **오프라인 탭**: 매장 위치, 영업시간, 편의시설, 오시는 길
- **온라인 탭**: 배송 정책, 결제 수단, CS 정보, 교환/환불 정책
- **브랜드/카테고리 탭**: 취급 브랜드, 주력 상품, 전문 카테고리
- **이미지 탭**: 상점 사진 갤러리 관리

##### 오프라인 매장 상세 정보
오프라인 매장의 특성에 맞춰 다양한 정보를 입력할 수 있습니다.

- 상세 위치 설명 (2층, 지하, 특정 구역 등)
- 점심시간 설정
- 편의시설 체크 (휠체어 접근성, 아동 동반 가능 여부)
- 대중교통 및 도보 경로 상세 안내
- 주차 정보 및 주차 요금

##### 온라인 쇼핑몰 상세 정보
온라인 쇼핑몰 운영에 필요한 모든 정보를 관리할 수 있습니다.

- 지역별 차등 배송비 설정
- 다양한 결제 수단 등록 (카드, 계좌이체, 간편결제 등)
- CS 채널별 정보 (전화, 카카오톡, 이메일)
- 교환/환불 정책 상세 설정
- 모바일 웹 지원 여부
- 당일배송 가능 여부

##### 복합 상점 특화 기능
온·오프라인을 함께 운영하는 상점을 위한 특별한 기능입니다.

- 매장 픽업 서비스 설정
- 온라인 주문 → 오프라인 수령 옵션
- 온·오프라인 재고 연동 관리

#### 🎨 브랜드 & 카테고리 관리

- 취급 브랜드를 검색하고 추가할 수 있는 편리한 UI
- 주력 브랜드를 설정하여 고객에게 강조 표시
- 토슈즈, 레오타드, 타이즈 등 카테고리별 전문성 표시
- 브랜드별 재고 상태 관리

#### 🖼️ 이미지 갤러리 시스템

- 여러 장의 상점 사진을 업로드하고 관리
- 드래그 앤 드롭으로 이미지 순서 변경
- 자동 썸네일 생성 및 이미지 최적화
- 고화질 이미지 지원 (5MB 제한)

---

### 💆 개선된 기능

#### 메인 화면 정보 표시 강화

- 상점 카드에 더 많은 정보 표시
- 상점 유형 뱃지 (오프라인/온라인/복합)
- 주요 취급 브랜드 표시
- 실시간 영업 상태 표시 (영업중/영업종료/점심시간)

#### 상세 화면 대폭 개선

- 새로운 '정보' 탭 추가로 모든 상점 정보를 한눈에 확인
- 이미지 갤러리 뷰어로 여러 상점 사진 확인
- 상점 유형별 맞춤 정보 표시
- 브랜드별 제품 목록 표시

#### 성능 최적화

- 이미지 로딩 속도 30% 향상
- 데이터 캐싱으로 빠른 화면 전환
- 배치 로딩으로 네트워크 사용량 감소
- UI 렌더링 최적화로 부드러운 사용 경험

## Version 1.0.0 - 2024년 12월 25일

### 🆕 새로운 브랜드 아이덴티티

#### 앱 이름 변경
- **발레플러스 (Ballet+)** 로 새롭게 출발합니다
- 더 많은 서비스를 더하겠다는 의미를 담았습니다

#### 새로운 앱 아이콘
- 발레의 우아함을 담은 새로운 디자인
- 파스텔 핑크톤의 감성적인 아이콘

### 🔄 자동 업데이트 시스템

#### 버전 관리 기능
앱을 항상 최신 상태로 유지할 수 있도록 자동 버전 체크 시스템을 도입했습니다.

- **선택적 업데이트**: 새로운 기능이 추가되면 알림을 받고 원할 때 업데이트
- **필수 업데이트**: 중요한 보안 패치나 필수 기능 업데이트 시 안내
- **서버 점검 안내**: 서비스 점검 시 사전 공지

#### Firebase 통합
- 실시간 버전 관리를 위한 Firebase Remote Config 적용
- 앱 사용 분석을 위한 Firebase Analytics 통합
- 안정적인 서비스 제공을 위한 인프라 구축
